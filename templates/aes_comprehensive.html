AES (Advanced Encryption Standard)

Comprehensive Explanation

AES (Advanced Encryption Standard) is a symmetric encryption method using a shared key for both encryption and decryption. It processes data in 128-bit blocks, with key sizes of 128, 192, or 256 bits.

Encryption Process:

  • SubBytes: Each byte is substituted using a complex S-Box, adding non-linearity.
  • ShiftRows: Each row in the matrix is cyclically shifted left, introducing diffusion.
  • MixColumns: Column values are transformed with a fixed matrix, further mixing the data.
  • AddRoundKey: Each byte is XORed with a round-specific key.
